# Запросы

Создание, удаление и редактирование как обычно. Остальные ниже.

## Аутунтификация:
```
/api/authenticate
```

## Добавить в избранное:
```
/api/articles/:id/add_to_favorites
```

## Список избранного:
```
/api/favorites
```

## Список авторов:
```
/api/show_authors
```

## Все новости конкретного автора:
```
/api/author/:id
```

## Непрочитанные новости:
```
/api/show_unread_articles
```
